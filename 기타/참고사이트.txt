5. 아이콘
    https://fontawesome.com/

    <script src="https://kit.fontawesome.com/ec2e6610a2.js"
    crossorigin="anonymous"></script>
    아이콘 쓰기위한 스크립트

    ex) <a href="#"><i class="fa-solid fa-house"></i> HOME</a>

6. 그래프

    https://www.chartjs.org/

7. 사진 슬라이드
    https://swiperjs.com/
    swipe
    get start 클릭
    cdn 복사 붙여넣기
    html 레이아웃 복사 붙여넣기
    json 복사해서 script 새로 만들어서 안에 넣기
    이미지 자동으로 넘어가는 기능 : json안에 autoplay 추가
        autoplay : {
            delay : 1000,
            // 딜레이 1초
        }
    스크롤바는 필요없어서 지움

8. j쿼리
    jquery 다운로드 구글에 검색, jQuery CDN 눌러서 다운로드 홈페이지 들어감
    https://releases.jquery.com/
    jQuery 3.x 의 minified 클릭

    압축해서 cdn으로 사용할 것임
    스크립트 복사하고 붙여넣어서 사용
