8. 제이쿼리(jQuery)
    https://releases.jquery.com/

    제이쿼리UI 
        minified의 cdn 복붙
        theme 주소 link로 연결 (제이쿼리UI 기능들에 해당하는 CSS를 적용하기 위함)

    Plugins - animation - jQuery UI Effects Core - Try a Demo 에서 제이쿼리UI의 기능을 확인해 볼 수 있음
        Datepicker 많이 사용함
        사용하고 싶으면 아래의 view source 클릭
