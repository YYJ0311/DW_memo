ajax(Asynchronous JavaScript And XML)
    Ajax는 비동기적인 웹 애플리케이션의 제작을 위해 아래와 같은 조합을 이용하는 웹 개발 기법이다.

    비동기라서 실행순서가 후순위로 밀림
        => 자바스크립트를 위에서 아래로 읽을 때 ajax 위와 아래를 먼저 읽고 ajax를 읽으므로, ajax 내부에서 실행된 코드를 ajax 아래에서 불러올 수 없다.
