```
JQuery
    on 
        : 이벤트처리, 바인딩, 익명함수
        $().on(이벤트,익명함수)
        이벤트가 발생하면 익명함수를 실행하라!
        ** 자바스크립트는 매개변수에 함수를 넣을 수 있다 ( 익명함수 자리에 이미 존재하는 다른 함수를 넣어도 된다)

    익명함수를 쓰는 이유 : 함수가 많아지면 함수 이름 짓기가 너무 힘들어서

    append & appendto
        : 추가 또는 이동기능
        "부모태그"에 append하면 "자식태그"로 추가된다.
        기존에 존재하는 태그는 이동되었음

    children & find
        : 부모태그를 기준으로 자식태그의 정보를 알고싶을 때 사용
        * 배열 형태로 가져왔다!
        자식태그 안에 자식태그(손자?태그)를 찾을 땐 사용 못 함 (==> 1. 자식태그를 부모태그로 해서 그 자식(손자)태그 찾음 2. find 사용)

    개발하는데 제일 오래 걸림
        자바는 큰 변동사항이 없지만 제이쿼리는 수정해야될 게 많다.(아마 함수 지정, 태그이름 지정된 걸 수정해야 돼서??)

    제이쿼리 파일을 보면 ~.min.js 라고 적혀있음
        min 의 의미 : 압축된 파일(엔터같은 데이터가 다 압축돼서 줄바꿈 없음)
```

```
내장함수

trim(), substring() 등은 문자열 함수이고 자바스크립트의 내장함수이다.
=> j쿼리는 다운받아서 썼지만 내장함수는 다운받지 않고 사용 가능함

var userId = $('#userId').val().trim(); 에서
1. val()은 제이쿼리에 있는 함수이고, 문자를 리턴함
2. trim()은 문자 자체에 내장돼 있는 함수
$('#userId').val()에서 return한 문자에 .trim() 함수 적용 
=> 메소드 체이닝
```
