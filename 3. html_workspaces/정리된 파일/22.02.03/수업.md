- head tag       
    1. title : 웹사이트의 제목(브라우저 상단의 페이지 이름)을 쓸 수 있음.      
    2. *** meta(메타) 데이터를 작성할 수 있음.     
        메타란? 어떤 목적을 가지고 만들어진 데이터      
        메타태그는 검색 결과를 알맞게 만들어 줌     
                  검색시 상위 노출시킬 때 사용
        <meta name="keywords" content="대전,술집,추천,대전술집,술집맛집" />
        <meta name="author" content="유영준" />
