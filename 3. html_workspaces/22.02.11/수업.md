2. padding, margin, box-sizing
    padding : 부모와 자식 간격 조절, 부모태그에 적는다.  
    margin : 부모와 부모 간격 조절, 부모태그에 적는다.  
    box-sizing : 박스크기 조절. 값을 border box로 해서 자식박스가 부모 박스를 넘어가지 않게 함
