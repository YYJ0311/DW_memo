SQL

        SELECT

        UPDATE(= DELETE 후 INSERT)      
        INSERT      

```sql    
SELECT
    컬럼이름
FROM 테이블
WHERE 조건식
GROUP BY 컬럼이름
HAVING 집계함수, 그룹핑된 컬럼들 조건
ORDER BY 컬럼이름 ASC
LIMIT

순서 : FROM -> WHERE -> GROUP BY -> HAVING -> SELECT -> ORDER BY -> LIMIT
```
