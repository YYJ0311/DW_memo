```
비즈니스 아키텍처
    비즈니스 모델 및 구조를 정의하며 비즈니스 모델을 위한 기능과 프로세스를 정의함

데이터 아키텍처

    전사적인 아키텍처 구축이 목적

```
스토리보드

    완성해야 할 서비스와 예상되는 사용자 경험을 미리 보기 위한 방법론
    실제 개발 단계에서 발생할 수 있는 문제점을 미리 발견하고 대처하기 위해 작성한다

```
UI 설계 용어
    Wireframe
        UI 중심의 화면 레이아웃을 설계하며, 선과 뼈대를 만드는 것으로 시각적인 프레임워크, 버튼 배치, 페이지 요소 등의 계층 구조 등 페이지 구조와 구조성 측면의 결과를 통합적으로 적용하는 작업이다.
        Wireframe을 작성하면 개발 구성원과 소통에 용이하며, 시각적인 문서작업, 업무의 방향성 그리고 화면이 어떻게 동작하는지에 대하여 이해를 높일 수 있다
        Wireframe 작성 3단계
            견적 : 예산, 작업일정 등의 대략적인 견적과 화면 흐름에 대한 계획
            기능 : 화면들을 상세하게 설께하여 리스트 업 하고 화면과 플로우에 맞게 컨트롤러들을 디자인 함
            유용성과 디자인 초안 : 사용성을 가다듬고 앱의 전체적인 Look을 결정하며 데이터 모델과 컨트롤러 레이아웃을 반복적으로 검토
    Mockup
        실물과 흡사한 정적인 형태의 모형
        서비스 개발 전에 결과물을 그려보는 것
    Prototype
        다양한 인터랙션을 결합하여 실제 서비스처럼 작동하는 동적인 형태의 모형을 작성하는 것
```

```
User Interface
    좋은 사용자 인터페이스는 심리학과 생리학에 기반하여 사용자가 필요로 하는 요소를 쉽게 찾고 사용하며, 그 요소로 명확하게 의도한 결과를 쉽게 얻어 낼 수 있어야 함
User Experience
    보기 좋은 시각물을 만드는 것보다 풍부한 사용자 경험을 제공하는 것이 목표
    양질의 사용자 경험을 제공하기 위해서 사용자를 이해하고 인지적, 감성적, 행위적 특성을 발견하여 사용자의 NEEDS를 도출한 디자인을 만들어 내야 함
    UX 설계 시 사용자가 어떻게 받아들이고, 어떤 상황에서, 무엇을 느끼며, 어떻게 반응하는지에 대한 관점을 파악해야 함

```
jQuery 태그요소 조작
    선택자
        선택한 요소들을 변수에 저장하여 사용 가능

        모든 요소 선택
            $("*")
        class intro의 모든 <p> 선택
            $("p.intro")
        첫번째 p 선택
            $("p:first")
        모든 <ul>의 첫번째 <li> 선택
            $("ul li:first-child")
        모든 <a>에서 target의 값이 _blank인 요소 선택
            $("a[target = '_blank']")
        <input>의 타입이 button이거나 <button>인 모든 요소 선택
            $(":button")
        <tr>의 모든 짝수/홀수 요소
            $("tr:even")
            $("tr:odd")

        필터링에 사용하는 선택자
            :eq(n)
                선택한 요소 중에서 인덱스가 n인 요소 선택
            :gt(n)
                인덱스가 n보다 큰 요소 모두 선택
            :lt(n)
                인덱스가 n보다 작은 요소 모두 선택
            :even / :odd
                인덱스가 짝수/홀수인 요소 모두 선택
            :first
                첫 번째 요소 선택
            :last
                마지막 요소 선택
            :animated
                애니메이션 효과가 실행 중인 요소 모두 선택
            :header
                h1~h6까지 요소 모두 선택
            :lang(언어)
                지정한 언어의 요소 모두 선택
            :not(선택자)
                지정한 선택자와 일치하지 않는 요소 모두 선택
            :root
                최상위 루트 요소 선택
            :target
                웹 페이지 URL의 fragment 식별자와 일치하는 요소 모두 선택
            :contains(텍스트)
                지정한 텍스트를 포함하는 요소 모두 선택
            :has(선택자)
                지정한 선택자와 일치하는 자손 요소를 갖는 요소 모두 선택
            :empty
                자식이 없는 요소 모두 선택
            :parent
                자식이 있는 요소 모두 선택
        
        input 요소를 선택하는 선택자
            :button, :checkbox, :file, :image, :password, :radio, :reset, :submit, :text, :checked, :selected,
            :input (<input>, <textarea>, <select>, <button> 요소를 모두 선택함), 
            :focus (현재 포커스된 요소), 
            :disabled / :enabled (비활성화/활성화된 요소 모두 선택)

        조상 요소 선택
            선택한 요소의 부모/조상 요소 선택
                .parent() / .parents()
            지정한 선택자에 해당하는 요소 이전까지 모두 선택
                .parentsUntil()
            조상요소 중에서 가장 첫 번째 요소 선택
                .closest()
        
        형제 요소
            형제 요소 모두 선택
                .siblings()
            선택한 요소 바로 다음에 위치한 형제 요소 선택 / 모두 선택
                .next() / nextAll()
            형제 요소 중에서 지정한 선택자 바로 이전까지의 요소 모두 선택
                .nextUntil()
            바로 이전에 위치한 형제 요소 선택 / 모두 선택
                .prev() / prevAll()
            지정한 선택자 바로 다음까지의 요소 모두 선택
                .prevUntil()

        자손 요소
            자식 요소 모두 선택
                children()
            자손 중에서 전달받은 선택자에 해당하는 요소 모두 선택
                find()

        그 외
            선택한 요소의 집합에 전달받은 요소 추가
                .add()
            선택한 요소의 집합에 바로 전에 선택했던 요소 추가
                .addBack()
            전달받은 콜백 함수를 실행
                .each()
            마지막으로 실행한 메소드의 실행 전 상태로 복원
                .end()
            자식 요소를 모두(텍스트와 주석까지 모두) 선택
                .contents()
    
    스타일
        클래스
            클래스 추가 제거
                .addClass() / removeClass()
            선택한 요소에 클래스가 없으면 인수로 전달받은 클래스를 추가하고, 전달받은 클래스가 추가돼 있으면 제거함
                .ToggleClass()
            인수로 전달받은 값이 선택한 요소의 클래스 이름과 일치하는지 확인
                .hasClass()
