<2교시>
### IT용어
디폴트 : 기본 값(기본으로 정해져 있는 것)

새로운 폴더에서 git을 쓰려면,
폴더 안 배쉬창에서 "git init" 한번만 입력

새폴더에서 git 파일을 다운받을 때,
폴더 안 배쉬창에서 "git clone 주소" 로 시작

git init을 한 폴더는 목적지를 정해줘야 해서
"git remote 주소" 를 입력해줘야 한다

만들어진 저장소(폴더)
git pull (주소 입력x) 로 서버에서 파일을 내려받고
작업 후, git add 로 작업한 파일을 저장
add 후, commit을 통해서 최종적으로 저장됨
퇴근하기 전에 push로 github에 업로드 한다.

출근하고 pull을 꼭! 하고 작업해야 나중에 push할 때 문제가 없다
pull->add->commit->push

---깃 저장소에 연결하지 않아도 git clone 주소 명령어로 파일을 다운받을 수 있다
