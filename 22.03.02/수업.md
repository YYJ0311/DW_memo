- IP가 유동적으로 바뀌는 것 = DHCP

- DHCP를 하는 이유
    : 제한된 인원이 꽉 찬 상태에서 나가면 다른 사람이 그 자리를 쓸 수 있음
    (학교, 회사, 카페 등등에서 사용)
    공유기 default 상태

- Static IP : 고정 아이피
    DHCP와 다르게 ip를 할당받고 나가도 여전히 할당된 상태로 다른 사람이 사용 불가능
    (만든 서비스를 제공할 때 사용(회원마다 하나씩 부여))
