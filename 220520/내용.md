동적 웹 : 프레임워크 + 템플릿엔진 으로 개발 (우리는 spring boot 와 jsp)

    1. 이미지(영상) 전용 서버로 많이 사용함
    2. 백업 서버로 사용
    3. 웹 호스팅 서버로 사용 (수업에서 사용)
